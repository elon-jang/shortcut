# Shortcut Pro - Legend Edition

키보드 단축키 학습을 위한 게이미피케이션 웹앱

## 기능

- **객관식 퀴즈**: 빠른 판단력 테스트
- **리얼 키 입력**: 실제 단축키를 눌러 근육 기억 훈련
- **플래시카드**: 자율 반복 암기

### 지원 카테고리
- VS Code
- macOS
- Chrome
- IntelliJ

### 게이미피케이션
- XP 시스템 및 레벨업
- 연속 학습 스트릭
- 콤보 보너스

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Lucide React

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```
