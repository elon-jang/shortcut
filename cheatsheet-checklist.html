Shortcut Memorization Tips

1
연상 기억법 (Mnemonics)

키와 기능을 연결하세요:

  • ⌘K = Kick off new chat
  • ⌘L = Location bar (주소창)
  • ⌘D = aDd bookmark / Duplicate
  • ⌘R = Refresh / Reload
  • ⌘W = close Window/tab
  • ⌘T = new Tab
2
Modifier 패턴 이해하기

수식 키의 일관된 의미:

  • ⌘ = 기본 동작
  • ⇧⌘ = 반대/확장 (예: Undo↔Redo)
  • ⌥ = 대체/고급 옵션
  • ⌃ = 시스템/터미널 레벨
3
그룹으로 학습하기

비슷한 단축키를 묶어서 암기:

  • 탭 이동: ⌃Tab / ⌃⇧Tab
  • 히스토리: ⌘[ / ⌘]
  • 분할: ⌘D / ⇧⌘D
4
하루 3개 규칙

매일 3개씩만 연습:

  1. 아침: 새 단축키 3개 선택
  2. 하루 종일: 의식적으로 사용
  3. 저녁: 체크박스 표시 ✓

80개 ÷ 3 = 약 4주면 마스터!

5
마우스 금지 챌린지

특정 작업에서 마우스 사용 금지:

  • 1주차: 탭 전환은 키보드로만
  • 2주차: 파일 열기는 ⌘P로만
  • 3주차: 창 관리는 Rectangle로만
6
손가락 근육 기억

눈 감고 10번씩 반복:

  1. 단축키를 보지 않고 누르기
  2. 틀리면 다시 10번
  3. 3일 연속 성공 = 완전 습득
